## Как работает

1. **Запуск:** При выборе уровня "Обучение" автоматически запускается tutorial
2. **Подсказки:** Черная полоса сверху экрана с текстом текущего шага
3. **Ограничения:** Клики только на разрешенные клетки
4. **Прогресс:** Автоматический переход к следующему шагу при выполнении условия
5. **Завершение:** Остановка tutorial при победе/поражении

## Технические особенности

- **Минимальный код:** Только необходимые изменения
- **Динамические клетки:** Поддержка генерируемых attack_cell
- **Автоматическая остановка:** При любом завершении забега
- **Блокировка кликов:** Неразрешенные действия игнорируются
- **Обновление состояния:** После каждого действия игрока
