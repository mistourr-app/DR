# Схема работы Tutorial системы

## Поток выполнения

```
┌─────────────────────────────────────────────────────────────┐
│                    Выбор уровня "Обучение"                  │
└────────────────────────────┬────────────────────────────────┘
                             │
                             ▼
┌─────────────────────────────────────────────────────────────┐
│  run.js: startRun('tutorial')                               │
│  ├─ Загрузка layout из registry.js                          │
│  └─ startTutorial() → tutorialActive = true, currentStep = 0│
└────────────────────────────┬────────────────────────────────┘
                             │
                             ▼
┌─────────────────────────────────────────────────────────────┐
│  renderer.js: renderRun()                                   │
│  └─ renderTutorialHint() → отображает подсказку шага        │
└────────────────────────────┬────────────────────────────────┘
                             │
                             ▼
┌─────────────────────────────────────────────────────────────┐
│  Игрок кликает на клетку (x, y)                             │
└────────────────────────────┬────────────────────────────────┘
                             │
                             ▼
┌─────────────────────────────────────────────────────────────┐
│  main.js: handleCanvasClick()                               │
│  └─ isClickAllowed(x, y) → проверка разрешенных клеток      │
│     ├─ false → клик игнорируется                            │
│     └─ true → processPlayerAction(x, y)                     │
└────────────────────────────┬────────────────────────────────┘
                             │
                             ▼
┌─────────────────────────────────────────────────────────────┐
│  run.js: processPlayerAction()                              │
│  ├─ Выполнение действия (движение, атака, подбор)           │
│  └─ updateTutorial() → проверка условия завершения шага     │
│     └─ если выполнено → currentStep++                       │
└────────────────────────────┬────────────────────────────────┘
                             │
                             ▼
┌─────────────────────────────────────────────────────────────┐
│  Цикл повторяется до завершения всех 12 шагов               │
└────────────────────────────┬────────────────────────────────┘
                             │
                             ▼
┌─────────────────────────────────────────────────────────────┐
│  combat.js: dealDamageToBoss() или dealDamageToPlayer()     │
│  └─ stopTutorial() → tutorialActive = false                 │
│     └─ Переход к RUN_VICTORY или RUN_SUMMARY                │
└─────────────────────────────────────────────────────────────┘
```

## Структура данных

```javascript
// tutorial.js
{
  tutorialActive: boolean,      // Активен ли tutorial
  currentStep: number,          // Текущий шаг (0-11)
  
  TUTORIAL_STEPS: [
    {
      text: "Подсказка для игрока",
      allowedCells: [{x, y}, ...] | null,  // Разрешенные клетки
      checkComplete: (state) => boolean    // Условие завершения
    },
    ...
  ]
}
```

## Пример шага

```javascript
// Шаг 3: Подбор здоровья
{
  text: "Подбери здоровье",
  allowedCells: [{x: 2, y: 3}],
  checkComplete: (state) => state.runState.player.pos.y >= 3
}

// Визуально:
┌─────────────────────────────┐
│   Подбери здоровье          │ ← Подсказка
├─────────────────────────────┤
│  [ ] [ ] [ ] [ ] [ ]        │
│  [ ] [ ] [ ] [ ] [ ]        │
│  [ ] [ ] [ ] [ ] [ ]        │
│  [ ] [ ][+6][ ] [ ]  ← y=3  │ ← Только эта клетка кликабельна
│  [ ] [ ] [ ] [ ] [ ]        │
│  [ ] [👤][ ] [ ] [ ]  ← y=0  │ ← Игрок
└─────────────────────────────┘
```

## Особые случаи

### Динамические клетки (шаг 11)

```javascript
{
  text: "Наступи на клетку атаки чтобы ударить босса",
  allowedCells: null,  // Определяется динамически!
  checkComplete: (state) => state.runState.boss.currentHp < state.runState.boss.hp
}

// В isClickAllowed():
if (currentStep === 10 && runState.levelPhase === 'boss_arena') {
  const cell = runState.rows[y]?.[x];
  return cell?.type === 'attack_cell';  // Только attack_cell
}
```

### Финальный шаг (шаг 12)

```javascript
{
  text: "Победи босса!",
  allowedCells: null,  // Любые клики разрешены
  checkComplete: (state) => state.runState.boss.currentHp <= 0
}

// В isClickAllowed():
if (currentStep === 11) {
  return true;  // Все клики разрешены
}
```

## Состояния tutorial

```
INACTIVE ──startTutorial()──> ACTIVE (step 0)
                                  │
                                  │ updateTutorial()
                                  ▼
                              ACTIVE (step 1)
                                  │
                                  │ ...
                                  ▼
                              ACTIVE (step 11)
                                  │
                                  │ stopTutorial()
                                  ▼
                              INACTIVE
```
