# 🚀 Быстрый старт Tutorial

## Запуск за 3 шага

### 1. Запустите сервер
```bash
node server.js
```

### 2. Откройте браузер
```
http://localhost:5500
```

### 3. Выберите уровень
Нажмите на кнопку **"Обучение (13 рядов)"**

---

### Что дальше?

После каждого действия подсказка автоматически меняется на следующий шаг.

Всего **12 шагов**, которые научат вас:
- Движению и прыжкам
- Подбору предметов
- Ближнему и дальнему бою
- Использованию бонусов
- Бою с боссом

## Если что-то не работает

### Проверьте консоль браузера
`F12` → вкладка Console

### Проверьте, что файлы на месте
```
DR/
├── main.js
├── renderer.js
├── combat.js
├── tutorial.js
└── ...
```

### Перезапустите сервер
```bash
Ctrl+C  (остановить)
node server.js  (запустить снова)
```
