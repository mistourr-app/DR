## 🏗️ Архитектура

```
tutorial.js (центральный модуль)
    ↓
    ├─→ main.js (проверка кликов)
    ├─→ renderer.js (отображение подсказок)
    ├─→ run.js (запуск, обновление, кастомные значения)
    ├─→ combat.js (остановка)
    ├─→ registry.js (layout уровня)
    └─→ bossAI.js (проверки на null)
```

### Поток данных

```
Выбор уровня "Обучение"
    ↓
startTutorial() → tutorialActive = true, currentStep = 0
    ↓
renderTutorialHint() → показать подсказку шага
    ↓
Клик игрока → isClickAllowed(x, y) → проверка
    ↓
processPlayerAction() → выполнение действия
    ↓
updateTutorial() → checkComplete() → currentStep++
    ↓
Повтор до currentStep >= 15
    ↓
stopTutorial() → tutorialActive = false
```
