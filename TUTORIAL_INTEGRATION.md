## Как запустить

```bash
node server.js
```

Откройте браузер: `http://localhost:5500`

Выберите уровень "Обучение (13 рядов)"

## Как работает

1. При выборе tutorial уровня автоматически запускается `startTutorial()`
2. На экране появляется черная полоса с подсказкой текущего шага
3. Игрок может кликать только на разрешенные клетки
4. После выполнения условия шага автоматически переход к следующему
5. При победе/поражении tutorial останавливается

## Визуальное отображение

- Подсказка: черная полупрозрачная полоса сверху экрана
- Текст: белый, жирный, 16px, по центру
- Фон: rgba(0, 0, 0, 0.7)
- Высота: 60px
