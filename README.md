# Dungeon Card Crawler

Мобильный пошаговый fast-paced dungeon crawler в жанре rogue-lite.

## Концепция

**Целевая аудитория:** Мужчины 18+, которые хотят получить опыт быстрого dungeon crawler на мобильных устройствах.

**Уникальная фишка:** Очень быстрый геймплей, близкий к раннерам, но с тактической глубиной.

**Технологии:** PWA app на чистом JavaScript с HTML5 Canvas.

## Архитектура

### Модули
- **main.js** - точка входа, игровой цикл, обработка кликов
- **state.js** - глобальное состояние (appState, runState, metaState)
- **config.js** - константы и настройки
- **run.js** - логика забега (движение, взаимодействия)
- **combat.js** - боевая система
- **enemyAI.js** - логика врагов (проверка видимости, атаки)
- **bossAI.js** - AI босса
- **renderer.js** - отрисовка игрового поля
- **ui.js** - UI экраны и элементы
- **animation.js** - система анимаций
- **events.js** - система событий
- **registry.js** - регистры врагов, клеток, уровней
- **utils.js** - утилиты (PRNG, генерация объектов)

### Состояния приложения
```
BOOT → META_HUB → RUN_PLAYING → RUN_SUMMARY/RUN_VICTORY → META_HUB
```

### Фазы уровня
- **dungeon** - забег по рядам вверх
- **boss_arena** - бой с боссом (последние 2 ряда)

## Разработка

### Запуск
```bash
node server.js
```
Игра доступна на `http://localhost:5500`

### Отладка
Используйте seed для воспроизводимости:
```
http://localhost:5500/?seed=12345
```

### Структура данных

**RunState:**
- player (hp, energy, pos, inventory, weapon)
- boss (hp, pos, inventory)
- rows (сетка клеток)
- levelPhase ('dungeon' | 'boss_arena')
- turnOwner ('player' | 'boss' | 'processing')
- seed (для генерации)

**MetaState:**
- currency (валюта)
- upgrades (апгрейды)
- unlocks (разблокировки)
