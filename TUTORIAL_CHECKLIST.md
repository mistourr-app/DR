# Чеклист тестирования Tutorial

## Запуск
- [ ] Запустить `node server.js`
- [ ] Открыть `http://localhost:5500`
- [ ] Выбрать уровень "Обучение (13 рядов)"

## Шаг 1: Движение вперед
- [ ] Видна подсказка: "Тапни на клетку впереди чтобы двигаться"
- [ ] Можно кликнуть только на клетку (2, 1)
- [ ] После клика игрок перемещается
- [ ] Подсказка меняется на следующий шаг

## Шаг 2: Движение в сторону
- [ ] Подсказка: "Тапни на клетку сбоку (стоит 1 энергию)"
- [ ] Можно кликнуть на (1, 2) или (3, 2)
- [ ] Энергия уменьшается на 1
- [ ] Игрок перемещается

## Шаг 3: Подбор здоровья
- [ ] Подсказка: "Подбери здоровье"
- [ ] Клетка (2, 3) содержит heal
- [ ] После клика HP увеличивается
- [ ] Всплывает зеленый текст "+6"

## Шаг 4: Подбор патронов
- [ ] Подсказка: "Подбери патроны для арбалета"
- [ ] Клетка (2, 4) содержит ammo
- [ ] Заряды увеличиваются на 2
- [ ] Всплывает оранжевый текст "+2 з."

## Шаг 5: Ближний бой
- [ ] Подсказка: "Атакуй врага в ближнем бою"
- [ ] Клетка (2, 5) содержит врага TYPE_2 (8 HP)
- [ ] Происходит взаимный обмен уроном
- [ ] Враг умирает, игрок получает урон

## Шаг 6: Дальний бой
- [ ] Подсказка: "Выстрели из арбалета во врага сбоку"
- [ ] Враги на (1, 6) и (3, 6)
- [ ] Видна белая пунктирная линия к врагу
- [ ] После выстрела враг получает урон
- [ ] Заряды уменьшаются на 1

## Шаг 7: Атака в спину
- [ ] Подсказка: "Иди вперед. Враг сзади атакует в спину!"
- [ ] Враг на (2, 7)
- [ ] При движении на (2, 8) враг атакует
- [ ] Игрок получает урон
- [ ] Красная вспышка экрана

## Шаг 8: Бонус атаки
- [ ] Подсказка: "Подбери бонус атаки"
- [ ] Клетка (2, 9) содержит attack_bonus
- [ ] Бонус появляется в инвентаре (желтый слот)
- [ ] Всплывает желтый текст "+5 атк."

## Шаг 9: Бой с бонусом
- [ ] Подсказка: "Атакуй врага с бонусом атаки"
- [ ] Враг на (2, 10)
- [ ] Урон = HP игрока + бонус атаки
- [ ] Бонус расходуется после атаки

## Шаг 10: Арена босса
- [ ] Подсказка: "Входим в арену босса! Двигайся горизонтально"
- [ ] Игрок на ряду 11 (арена)
- [ ] Можно двигаться только горизонтально
- [ ] Босс виден на ряду 11-12 (высокая карточка)

## Шаг 11: Клетка атаки
- [ ] Подсказка: "Наступи на клетку атаки чтобы ударить босса"
- [ ] На арене генерируются attack_cell (красные)
- [ ] Можно кликнуть только на attack_cell
- [ ] Босс получает урон
- [ ] Босс делает ответный ход

## Шаг 12: Победа
- [ ] Подсказка: "Победи босса!"
- [ ] Можно использовать любые действия
- [ ] При HP босса = 0 появляется экран победы
- [ ] Tutorial останавливается

## Дополнительные проверки
- [ ] Клики на неразрешенные клетки игнорируются
- [ ] Подсказка всегда видна сверху экрана
- [ ] При поражении tutorial останавливается
- [ ] Можно перезапустить уровень
- [ ] После tutorial можно играть в другие уровни

## Визуальные элементы
- [ ] Черная полоса с подсказкой (60px высота)
- [ ] Белый текст по центру
- [ ] Подсказка не перекрывает игровое поле
- [ ] Подсказка читаема на всех разрешениях
