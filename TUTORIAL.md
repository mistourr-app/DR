# Обучающий уровень (Tutorial)

## Реализация

Обучающий уровень состоит из 12 шагов, которые последовательно обучают игрока всем механикам игры.

### Архитектура

**tutorial.js** - основной модуль:
- `TUTORIAL_STEPS` - массив шагов с текстом подсказки и условиями завершения
- `startTutorial()` - запускает обучение
- `stopTutorial()` - останавливает обучение
- `updateTutorial()` - проверяет выполнение текущего шага
- `isClickAllowed(x, y)` - проверяет, разрешен ли клик на клетку
- `getCurrentStep()` - возвращает текущий шаг

### Интеграция

1. **registry.js** - уровень 'tutorial' с предопределенным layout
2. **run.js** - запуск tutorial при старте уровня, обновление после действий
3. **main.js** - проверка разрешенных кликов
4. **renderer.js** - отображение подсказок поверх игрового поля
5. **combat.js** - остановка tutorial при победе/поражении

### Шаги обучения

1. **Движение вперед** - клик на клетку впереди (y=1)
2. **Движение в сторону** - клик на боковую клетку за энергию (y=2)
3. **Подбор здоровья** - клик на heal (y=3)
4. **Подбор патронов** - клик на ammo (y=4)
5. **Ближний бой** - атака врага в упор (y=5)
6. **Дальний бой** - выстрел из арбалета (y=6)
7. **Атака в спину** - враг атакует при уходе с ряда (y=8)
8. **Бонус атаки** - подбор attack_bonus (y=9)
9. **Бой с бонусом** - атака врага с бонусом (y=10)
10. **Арена босса** - горизонтальное движение (y=11)
11. **Клетка атаки** - использование attack_cell
12. **Победа над боссом** - финальный бой

### Layout уровня

```javascript
// 13 рядов (0-12), последние 2 - арена босса
[
  [{}, {}, {}, {}, {}],                                    // 0: старт
  [{}, {}, {}, {}, {}],                                    // 1: пустой
  [{type: 'wall'}, {}, {}, {}, {type: 'wall'}],           // 2: стены
  [{}, {}, {type: 'heal'}, {}, {}],                       // 3: здоровье
  [{}, {}, {type: 'ammo'}, {}, {}],                       // 4: патроны
  [{}, {}, {type: 'enemy', enemyType: 'TYPE_2'}, {}, {}], // 5: враг
  [{type: 'enemy', enemyType: 'TYPE_1'}, {}, {}, {type: 'enemy', enemyType: 'TYPE_1'}, {}], // 6: враги сбоку
  [{}, {}, {type: 'enemy', enemyType: 'TYPE_2'}, {}, {}], // 7: враг сзади
  [{}, {}, {}, {}, {}],                                    // 8: пустой
  [{}, {}, {type: 'attack_bonus'}, {}, {}],               // 9: бонус атаки
  [{}, {}, {type: 'enemy', enemyType: 'TYPE_2'}, {}, {}], // 10: враг
  [{}, {}, {}, {}, {}],                                    // 11-12: арена босса
  [{}, {}, {}, {}, {}]
]
```

### Особенности

- Подсказки отображаются в черной полупрозрачной полосе сверху экрана
- Клики на неразрешенные клетки игнорируются
- Для шагов 10-11 (боссфайт) клетки определяются динамически
- Tutorial автоматически останавливается при победе или поражении
