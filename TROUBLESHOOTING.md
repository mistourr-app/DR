# Решение проблем

## Черный экран при запуске

### Проблема: 404 Not Found для файлов

**Решение 1: Жесткая перезагрузка**
```
Ctrl + Shift + R  (или Ctrl + F5)
```

**Решение 2: Очистить кэш браузера**
1. F12 → вкладка Network
2. Правый клик → Clear browser cache
3. Перезагрузить страницу

**Решение 3: Проверить файлы напрямую**
Откройте в браузере:
- http://localhost:5500/config.js
- http://localhost:5500/main.js
- http://localhost:5500/tutorial.js

Если файлы открываются - проблема в кэше.
Если 404 - проблема в сервере.

### Проблема: Порт занят (EADDRINUSE)

**Решение:**
```bash
# Найти процесс
netstat -ano | findstr :5500

# Остановить процесс (замените PID на ваш)
taskkill /F /PID 4216

# Запустить сервер снова
node server.js
```

### Проблема: Модули не загружаются

**Проверьте консоль браузера (F12):**
- Ошибки импорта?
- Синтаксические ошибки?
- CORS ошибки?

**Решение:**
Убедитесь, что сервер запущен и отдает файлы с правильным MIME-типом:
```javascript
// В server.js должно быть:
'.js': 'application/javascript'
```

## Быстрая диагностика

### 1. Проверить сервер
```bash
node server.js
```
Должно быть: `Server running at http://127.0.0.1:5500`

### 2. Проверить браузер
Откройте: http://localhost:5500

### 3. Проверить консоль (F12)
Не должно быть красных ошибок

### 4. Проверить файлы
```bash
dir *.js
```
Должны быть: main.js, config.js, tutorial.js и др.

## Если ничего не помогает

1. Остановите все процессы Node.js
2. Удалите кэш браузера полностью
3. Перезапустите браузер
4. Запустите сервер заново
5. Откройте в режиме инкогнито

## Контакты для помощи

Если проблема не решается, проверьте:
- Версию Node.js: `node --version` (должна быть 14+)
- Права доступа к файлам
- Антивирус не блокирует порт 5500
